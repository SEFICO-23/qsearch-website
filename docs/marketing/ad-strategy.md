## 2. Platform Strategy

### Facebook — Reach + Fear + B2B Teaser

**Why:** Highest reach in Italian SMB market. Marco (titolare PMI) spends time here. Groups like "Imprenditori Italiani", "PMI Digitali" are active. Facebook Ads Manager has the best geo-targeting for Italian provinces.

**Strategy:**
- **Objective:** Lead Generation (B2B track) + Traffic (B2C signup)
- **Format:** Single image + carousel (2-3 slides showing scan → results → action)
- **Frequency cap:** 3 impressions/week per user (avoid ad fatigue)
- **Retargeting:** Pixel on qsearch.ch → retarget visitors who didn't convert after 3 days
- **Lookalike:** Upload customer list (when available) → 1% lookalike in Italy

**Estimated Performance (Italian market):**
| Metric | Estimate |
|--------|----------|
| CPM | €4–8 (Italy B2B) |
| CPC | €0.40–1.20 |
| CTR | 1.2–2.5% |
| Conversion rate (landing → scan) | 3–8% |
| Cost per lead (B2B scan) | €5–15 |

### LinkedIn — Authority + B2B + Compliance

**Why:** Laura (IT Manager), Davide (CTO), Alessandro (DPO) live here. LinkedIn targeting by job title + company size + industry is unmatched. Higher CPCs but dramatically higher intent.

**Strategy:**
- **Objective:** Lead Generation (B2B track) + Brand Awareness (thought leadership)
- **Format:** Sponsored Content (single image + document ads) + Sponsored InMail for high-value prospects
- **Content mix:** 60% value content (insights, stats, mini-reports) / 40% direct CTA
- **Company page:** Post 2-3x/week — mix of findings (anonymized), threat intel, platform updates
- **InMail:** Reserved for Persona 4 (compliance) — highest conversion, lowest volume

**Estimated Performance:**
| Metric | Estimate |
|--------|----------|
| CPM | €25–50 (Italy/CH B2B) |
| CPC | €3–8 |
| CTR | 0.4–0.9% |
| InMail open rate | 45–55% |
| InMail CTR | 3–5% |
| Cost per lead | €15–40 |

### Instagram — Brand + Visual Authority + Startup/Tech

**Why:** Davide (startup founder) and younger IT professionals are here. Visual-first platform lets us showcase the Ultraviolet Volt brand aesthetic. Stories and Reels have high engagement for tech content in Italy.

**Strategy:**
- **Objective:** Brand Awareness + Traffic (B2C signup)
- **Format:** Stories (swipe-up to scan) + Reels (terminal aesthetics, scanning demos) + Feed (data visualizations, threat stats)
- **Aesthetic:** Dark mode (Abyss background), terminal-style text, scanning animations, Volt Lime accents
- **Hashtags:** #cybersecurity #sicurezzainformatica #PMI #startup #hacking #NIS2 #GDPR #infosec
- **Collab posts:** With Italian tech influencers (see influencer-outreach.md)

**Estimated Performance:**

|--------|----------|

| CPC | €0.30–0.90 |
| CTR (Stories) | 0.5–1.5% |
| CTR (Reels) | 1–3% |

## 4. Conversion Funnels

### Funnel A: B2B Track (Fear → Scan → Pay)

```
Ad (fear-based) → qsearch.ch/scan
  → Enter company email + domain
    → Automatic T0 passive scan (free)
      → Teaser dashboard (risk score, redacted findings)
        → "Sblocca il report completo" CTA
          → Stripe checkout (CHF 190–6,500+)
            → Full report delivered
```

**Key metric:** Cost per paid report = Target < CHF 50

### Funnel B: B2C Track (Value → Try → Subscribe)

```
Ad (value-based) → qsearch.ch/pricing or /signup
  → Free account (3 credits)
    → First scan (quick scan, 1 credit)
      → See "21 findings locked" blur wall
        → Upgrade to Plus (CHF 19/mo)
          → Full access, more credits
```

**Key metric:** Cost per paid subscriber = Target < CHF 25

### Funnel C: Compliance Track (LinkedIn → InMail → Call)

```
LinkedIn content (compliance insights) → profile visit
  → InMail with compliance-specific pitch
    → qsearch.ch/scan (B2B track)
      → Teaser + "Schedule a call" (High/Critical tier)
        → Sales call → engagement
```

**Key metric:** Cost per qualified lead = Target < CHF 80

## 6. A/B Testing Plan

### Month 1: Creative Testing

| Test | Variants | Metric | Duration |
|------|----------|--------|----------|
| Fear vs. Value copy | "La tua azienda è protetta?" vs. "Scansione gratuita in 60 secondi" | CTR + conversion | 2 weeks |
| Dark vs. light creative | Ultraviolet Volt (dark) vs. Forest Gradient (light) | CTR + engagement | 2 weeks |
| CTA text | "Scansiona Ora" vs. "Scopri i Rischi" vs. "Prova Gratis" | Conversion rate | 2 weeks |
| Image vs. no-image | Terminal screenshot vs. clean text-only | CTR | 1 week |

### Month 2: Audience Testing

| Test | Variants | Metric | Duration |
|------|----------|--------|----------|
| Geo targeting | Lombardia vs. Piemonte vs. Veneto vs. Ticino | CPL by region | 3 weeks |
| Job title targeting (LI) | IT Manager vs. CTO vs. CEO vs. DPO | CPL + conversion quality | 3 weeks |
| Company size | 10–50 vs. 51–200 vs. 201–500 employees | CPL + deal size | 3 weeks |
| Age bracket | 25–35 vs. 36–45 vs. 46–55 | Engagement + conversion | 2 weeks |

### Month 3: Funnel Testing

| Test | Variants | Metric | Duration |
|------|----------|--------|----------|
| Landing page | Direct to /scan vs. /pricing vs. homepage | Conversion rate | 3 weeks |
| B2B teaser reveal level | Show 1 finding vs. 2 findings vs. score only | Paid conversion | 3 weeks |
| Free tier credits | 3 credits vs. 5 credits | Free → paid conversion | 4 weeks |
| Retargeting delay | 1 day vs. 3 days vs. 7 days | ROAS | 3 weeks |

### Testing Rules

1. **Minimum sample:** 1,000 impressions per variant before concluding
2. **Statistical significance:** 95% confidence before declaring winner
3. **One variable at a time:** Never test copy + creative + audience simultaneously
4. **Kill fast:** If a variant is 50%+ worse after 500 impressions, kill it early
5. **Document everything:** Results logged in this folder as `ab-test-results-YYYY-MM.md`
